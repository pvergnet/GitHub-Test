Content

Clone repository for new project : git clone {SSH URL}

Adding a file to git : git add test.html

adding all files to git : git add .

commit the changes on file : gt commit -m "My commit message"

Push the changes to github project : git push

Checking differences between commits : git diff

Create a branch for development : git checkout -b store

Push changes from a new branch : git push --set-upstream origin store

Switch branches : git checkout master

Merge files of store branch from main branch : git merge store
